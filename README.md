# MephiSem1Lab6
### Лабораторная работа № 6 «Работа с массивами структур. Исследование методов сортировки массивов»

В ходе выполнения лабораторной работы должна быть разработана программа, осуществляющая генерацию последовательностей данных, представленных в виде в виде массивов структур, и таймирование процессов их сортировки.
Программа должна реализовывать поддержку работы с тремя алгоритмами сортировок: с двумя из состава индивидуального задания и с реализацией алгоритма быстрой сортировки из состава стандартной библиотеки — функцией qsort().
При запуске программы пользователь должен иметь возможность указать через аргументы командной строки (флаги + позиционные аргументы):
1. алгоритм сортировки (один из трёх);
2. поле структуры, по которому должна осуществляться сортировка;
3. направление сортировки (по убыванию или по возрастанию);
4. количество элементов в генерируемых массивах;
5. количество генерируемых массивов.
Примечания:
1. Для работы с данными, формат которых описан в индивидуальном задании, должен быть разработан собственный составной тип данных — структура.
2. Логически законченные части алгоритма решения задачи должны быть оформлены в виде отдельных функций с параметрами. Использование глобальных переменных не допускается.
3. Для разбора аргументов командной строки должна применяться функция getopt() или её альтернативы.
4. Исходные коды программы должны быть логичным образом разбиты на несколько файлов.
5. Программа должна корректным образом работать с памятью, для проверки необходимо использовать соответствующие программные средства, например: valgrind (при тестировании и отладке программы ее необходимо запускать командой вида valgrind ./lab6, а при анализе производительности — ./lab6).

### Индивидуальное задание
**Структура данных**
Избиратель:

• ФИО (строка произвольной длины);

• номер избирательного участка (строка длиной 7 символов вида XXX-YYY, где X — это буква, а Y — цифра);

• возраст (целое число).

**Алгоритмы сортировки**
1. Сортировка расчёской (Comb sort).
2. Пирамидальная сортировка (Heap sort).
